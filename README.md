# ☁️ Cloud-Native Wallet LAB

> **Portfolio project** — A deliberately scoped digital wallet demonstrating
> cloud-native patterns: event-driven architecture, serverless AWS,
> infrastructure as code, and enterprise observability.

> Serverless digital wallet built with **NestJS**, **Next.js**, and **AWS**
> (Lambda, DynamoDB, EventBridge, SQS). Event-driven architecture with
> infrastructure as code (Terraform) and enterprise observability (Datadog).

> ⚠️ _This is a showcase of architecture and code quality, not a production wallet._
